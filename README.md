# 🧮 Basic Calculator

## 📌 Project Title
Basic Calculator using HTML, CSS, and JavaScript

------------------------------------------------------------

## 📖 About the Project
This project is a simple and interactive calculator built using HTML, CSS, and JavaScript.

It performs basic arithmetic operations such as:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus (%)

The calculator uses CSS Grid for proper button alignment and JavaScript for handling user input and calculations.

------------------------------------------------------------

## 🚀 Features
- Interactive Calculator Interface
- Display Screen for Input and Results
- Addition, Subtraction, Multiplication, Division
- Clear (C) Button
- Delete (DEL) Button
- Error Handling for Invalid Expressions
- Responsive Design
- CSS Grid Layout for Buttons

------------------------------------------------------------

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation, Event Handling, Operators)

## ▶️ How to Run the Project
1. Download or clone this repository
2. Make sure all three files are in the same folder:
   - index.html
   - style.css
   - script.js
3. Open `index.html` in any web browser
4. Start performing calculations
